# Hi, I'm Bader

I build full-stack web platforms and use DevOps practices to make them reliable, reproducible, and production-minded.

My work connects product engineering with infrastructure through clean user workflows, secure backend APIs, Dockerized environments, CI/CD automation, infrastructure-as-code, and observability. I care about building systems that are practical to use, easy to run, and clear for other developers to understand.
